About Me

Hi, im Ellie-Mae

I am particularly interested in electronic engineering, especially the design of CPU microcircuits, and I hope to pursue this to a PhD level. I am motivated by challenging problems and enjoy pushing myself to learn concepts beyond the standard curriculum.

Projects

RayCasting Utility Toolkit -OpenGL Extension-

RCUT (Raycasting Utility Toolkit) is an API built on top of OpenGL's GLUT. It was designed for developers—including myself—who want to create raytraced-style games more easily. The toolkit abstracts away much of the backend work typically required for graphics programming, allowing developers to focus primarily on building the game itself. In that sense, RCUT functions similarly to a game engine like Unity, but without the full UI and editor components.
